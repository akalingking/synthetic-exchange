# Synthetic Exchange #

Agent based generated timeseries data for simulating market conditions.

## Install ##
```
python -m pip install -r requirements.txt

or

python setup.py install
```

## Test ##
```
./scripts/run_ut.sh
```

## Run Synthetic Data ##
```
./bin/run_app
```
